# Craft Box

**一款专为 Minecraft PE 0.14.3 设计的开发工具箱。**

</div>

---

##  功能

*   **贴图转换** Tga和png贴图互相转换。
*   **图文教程文档** 在线更新的开发图文教程,支持md语法
*   **材质包转换**：将高版本JE或BE材质包转换为0.14可用的tga贴图包。
*   **JSON UI编辑器**：简易的JSON UI编辑器,支持语法高亮,主题切换,语法纠错,格式化,压缩成行。
*   **SO Patch**：so补丁管理,一键式应用补丁修补so。
*   **贴图编辑器**：像素级编辑贴图,支持多画布。
*   **自动化处理**：配套提供自动化的 APK 签名并静默安装(需要shizuku)工作流，提升开发效率。
*   ....等等一系列功能(其实多半还是大饼未实现)

##  构建

1.  **环境配置**：确保已安装Android Studio 以及 JDK 17+。
2.  **克隆仓库**：
    ```bash
    git clone https://github.com/hsh114514/Craft-Box.git
    ```

3.  **构建 APK**：使用Android Studio构建即可。
